
##第2章 2.2 创建入口文件

**创建入口文件main.go：**

 1. 需要import 你自己的配置文件包与cellgo框架
 2. 调用配置包中的配置项
 3. 项目运行全局函数cellgo.Run()用来初始化框架

    package main
    
    import (
    	"conf"
    
    	"github.com/mrkt/cellgo"
    )
    
    func main() {
    	conf.SetController()
    	cellgo.Run()
    }
